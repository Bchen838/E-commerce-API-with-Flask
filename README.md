# E-Commerce API (Flask, SQLAlchemy, Marshmallow, and MySQL)
The API supports full CRUD operations and allows management of users, products, and orders, including relational data handling through one to many and many to many associations. 

## Installation and Setup
1. Clone the repository
2. Create and activate virtual environment
3. Install dependendecied
4. Create a database

### Testing 
Tested using Postman and MySQL Workbench
